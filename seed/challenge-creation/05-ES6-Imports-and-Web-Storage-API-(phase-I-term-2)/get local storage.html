Storage.getItem()

The getItem() method of the Storage interface, when passed a key name, will return that key's value.

var aValue = storage.getItem(keyName);

Example

The following gets two data items inside local storage.

var currentColor = localStorage.getItem("bgcolor"); var currentFont = localStorage.getItem("font");

Values can also be set by using array notation.

var currentColor = localStorage["bgcolor"]; var currentFont = localStorage["font"];
